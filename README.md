# NotificationTimer

Type in the time you want, control it from the notification or use always screen on option to always keep an eyes on the reaminig time.<br>
Once the timer done, a simple notification will occurs.<br>
You can add an extra notification to get notified few seconds before the end to get ready.

## Features

* Timer controls from the notification
* Quick preset timer selection
* Big number display
* Always screen on
* Multi-window support
* Timer number to track laps
* Customizable background color progress bar
* Multi-windows support
